# AWS-Jenkins

1. Go to **AWS Marketplace** - **Discover products** - Search *Jenkins*
2. **Continue to Subsribe** **Accept Terms**
3. **Continue to Configuration**
4. **Continue to Launch**
5. **Review setings including EC2 and Key Pair Settings**
6. **Launch**
